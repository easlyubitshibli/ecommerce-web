# 🛒 ShopSmart – E-Commerce Platform for Electronic Products

ShopSmart is a full-stack e-commerce web application designed for buying and selling electronic products such as smartphones, tablets, smartwatches, and accessories. It provides a modern, secure, and user-friendly shopping experience with real-time product management and online payment integration.
--- 
## 🚀 Features

### 👤 User Features
- User registration & login
- Browse products by category
- Search and filter products
- View detailed product information
- Add/remove items from cart
- Place orders
- Secure online payment (SSLCommerz)

### 🛠️ Admin Features
- Add, update, delete products
- Manage inventory & stock
- View and manage orders
- User management system
- Dashboard for monitoring system activity

---

## 🧑‍💻 Tech Stack

**Frontend:**
- React.js (Vite)
- HTML, CSS, JavaScript

**Backend:**
- Node.js
- Express.js

**Database:**
- MongoDB (MongoDB Atlas)

**Payment Gateway:**
- SSLCommerz (Sandbox Mode)

**Tools & Others:**
- Git & GitHub
- VS Code
- RESTful APIs

---

## 🏗️ System Architecture

The system follows a **client-server architecture**:

- Frontend (React) communicates with backend via REST APIs
- Backend handles business logic and authentication
- MongoDB stores user, product, and order data
- SSLCommerz handles secure payment processing
